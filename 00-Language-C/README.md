## 1ª Parte do trabalho:

# Enunciado

A primeira parte do trabalho consiste em cada estudante **escolher** e **fazer** um estudo de uma linguagem de programação **compilada** ou seja, abordar os seguintes aspectos da linguagem escolhida como:

* Quem criou a linguagem (ano, país, etc)
* Tipos de compiladores da linguagem (criadores, ano, país)
* Vantagens e desvantagens entre os compiladores da linguagem
* Quais os IDEs da linguagem escolhida
* Tipos de dados da linguagem
* Algumas palavras reservadas
* Estrutura de controlo (seleção, repetição)
* Variáveis compostas homogéneas
* Variáveis compostas heterogéneas
* Função e procedimento (qual é a diferença na linguagem escolhida)
* Classe se tiver na linguagem
* Qual é o percentual de uso da linguagem a nível académico
* Qual é o percentual de uso da linguagem a nível empresarial
* Quais são os grandes ganhos da linguagem
* Qual é o percentual de uso da linguagem na programação em vídeo games
* Porque você recomendaria essa linguagem para uma pessoa
* Qual é o impacto da linguagem escolhida na informática no geral

# Observações:
* A entrega consiste de defesa (apresentação) na data previamente marcada em horário de aula.
